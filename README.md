# Ship_detection
🚢 ShipVision SAR: All-Weather Maritime IntelligenceShipVision SAR is a high-performance, ultra-lightweight deep learning solution designed to detect maritime vessels in Synthetic Aperture Radar (SAR) imagery. Unlike traditional optical satellite imagery, which is limited by darkness and cloud cover, SAR provides 24/7, all-weather monitoring capabilities. 🚀 Live DemoAccess the deployed web application here:(https://shipdetection-46akxvysxtruqqqjjrfcxs.streamlit.app/) 🎯 Key Performance MetricsOur model, built on the YOLOv8-Nano architecture, achieved elite results during validation:MetricValuemAP@50:98.6% Precision:97.6% Recall:95.2% Inference Speed:3.9 ms (on Tesla T4) Model Size:6.2 MB 📊 Dataset Used: SSDDThe model was trained and validated using the SSDD (SAR Ship Detection Dataset).Content: 1,160 SAR images containing 2,456 ships.Scenarios: Includes both Inshore (complex port environments with heavy land clutter) and Offshore (open sea) imagery.Sensors: Data sourced from various radar satellites including RadarSat-2, TerraSAR-X, and Sentinel-1.Significance: SSDD is the benchmark dataset for SAR ship detection, providing the diversity needed to ensure model robustness against "Speckle Noise. "🛠️ Technical MethodologyPreprocessing: Applied radar backscatter normalization and speckle noise reduction.Architecture: Utilized YOLOv8n (Nano) for its optimal balance of speed and accuracy.Training: Trained for 50 epochs using CIoU Loss and Mosaic Augmentation.Deployment: Built a user-friendly dashboard using Streamlit for real-time vessel counting and visualization.
